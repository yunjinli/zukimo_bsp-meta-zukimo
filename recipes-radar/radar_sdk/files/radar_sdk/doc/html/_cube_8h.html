◆ IFX_CUBE_AT

#define IFX_CUBE_AT (   cub,
  r,
  c,
  s 
)    (IFX_CUBE_DAT(cub)[IFX_CUBE_OFFSET(cub, r, c, s)])

Access cube element.

The macro can be used to set and get elements of a cube, for example:

foo = IFX_CUBE_AT(cube, row, col, slice);
IFX_CUBE_AT(cube, row, col, slice) = bar;

The macro works with both real (ifx_Cube_R_t) and complex (ifx_Cube_C_t) cubes.
