◆ ifx_avian_cw_capture_frame()

IFX_DLL_PUBLIC ifx_Matrix_R_t* ifx_avian_cw_capture_frame ( ifx_Avian_CW_Control_Handle_t  handle,
ifx_Matrix_R_t *  frame 
)

This method captures one frame of raw data.

This function captures for all activated antennas samples_per_antenna and returns the samples as a matrix with dimensions num_antennas (rows) x samples_per_antenna (columns).

The samples of the raw data are normalized to values between -1...1. To convert the samples to voltages, each sample must be multiplied by the reference voltage 1.21V.

If frame is NULL, memory for the matrix will be allocated and returned. Otherwise the memory of frame will be used.

Parameters
[in]handleA handle to the cw controller
[in]framePointer to the ifx_Matrix_R_t frame where raw data is stored. If this is NULL, then a new frame is created. the caller is responsible to deallocate the memory.
Returns
pointer to the ifx_Matrix_R_t frame containing the received samples.

◆ ifx_avian_cw_create()

IFX_DLL_PUBLIC ifx_Avian_CW_Control_Handle_t ifx_avian_cw_create ( ifx_Avian_Device_t *  handle)

This function creates and initializes the constant wave mode controller instance.

The provided port within the device handle is used to detect the sensor type of the connected device. According to the detected sensor type the CW mode controller is initialized. If the detected device is not recognized, an error is set.

Corresponding destroy method is not provided for CW mode. Reason is CW controller is part of device handle. In order to destroy a CW controller, application must destroy device handle.

After initialization the connection is still in deep sleep mode.

Parameters
[in]handleA device handle created before calling ifx_avian_cw_create
Returns
A handle to the CW controller, used to control device for CW mode

)

This method sets the number of samples per Rx antenna to be captured in single shot.

All active Rx antennas have same number of samples. There is no need to pass antenna index.

The number of samples must be an even number. If samples is an odd number the error IFX_ERROR_ARGUMENT_INVALID is set.

Parameters
[in]handleA handle to the cw controller
[in]samplesThe number of samples per Rx antenna to be captured in one shot
