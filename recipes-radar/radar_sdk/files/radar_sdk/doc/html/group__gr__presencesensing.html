Performs the presence sensing algorithm on given raw data.

Perform the presence sensing algorithm on the given matrix consisting of raw IF frame data coming from the configured radar transceiver.

The matrix frame_Data consists of IF frame data from the radar transceiver. The number of rows of the input matrix should match the number of chirps per frame and the number of columns should match the number of samples per chirp expected by the ifx_PresenceSensing_t (given in the ifx_Avian_Config_t of the ifx_PresenceSensing_Config_t used while creating the given presence sensing handle).

Parameters
[in]handleHandle to the presence sensing object.
[in]frame_dataMatrix consisting of IF frame data from the radar transceiver.
[out]resultStructure containing the result of the run.
