enum ifx_Avian_ADC_Oversampling

This enumeration type lists the available ADC oversampling modes.

The Avian device's internal Analog-Digital-Converter (ADC) is capable to repeat the full sample-hold-convert cycle multiple times and return the average of all cycles.

Enumerator
IFX_ADC_OVERSAMPLING_OFF 

No oversampling.

IFX_ADC_OVERSAMPLING_2x 

Oversampling factor 2.

IFX_ADC_OVERSAMPLING_4x 

Oversampling factor 4.

IFX_ADC_OVERSAMPLING_8x 

Oversampling factor 8.

◆ ifx_Avian_ADC_SampleTime

enum ifx_Avian_ADC_SampleTime

This enumeration type lists the available ADC sample time settings.

The sample time is the time that the sample-and-hold-circuitry of the Avian device's Analog-Digital-Converter (ADC) takes to sample the voltage at it's input.

Note
The specified timings refer to an external reference clock frequency of 80MHz. If the clock frequency differs from that, the sample time periods are scaled accordingly.
Enumerator
IFX_ADC_SAMPLETIME_50NS 

The voltage is sampled for 50ns.

IFX_ADC_SAMPLETIME_100NS 

The voltage is sampled for 100ns.

IFX_ADC_SAMPLETIME_200NS 

The voltage is sampled for 200ns.

IFX_ADC_SAMPLETIME_400NS 

The voltage is sampled for 400ns.

◆ ifx_Avian_ADC_Tracking

enum ifx_Avian_ADC_Tracking

This enumeration type lists the available ADC tracking modes.

The Avian device's internal Analog-Digital-Converter (ADC) has 11 bit resolution and a 12 bit result register. A single conversion always produces result values with unset LSB. The resolution can be increased by performing additional tracking conversion. The result will be the average of all conversions.

Enumerator
IFX_ADC_NO_SUBCONVERSIONS 

Each sampled voltage value is converted once.

IFX_ADC_1_SUBCONVERSIONS 

Each sampled voltage value is converted two times.

IFX_ADC_3_SUBCONVERSIONS 

Each sampled voltage value is converted four times.

IFX_ADC_7_SUBCONVERSIONS 

Each sampled voltage value is converted eight times.

◆ ifx_Avian_Test_Signal_Generator_Mode

enum ifx_Avian_Test_Signal_Generator_Mode

This type enumerates the available modes of the Avian device's test signal generator.

Enumerator
IFX_TEST_SIGNAL_MODE_OFF 

The generator is not used.

IFX_TEST_SIGNAL_MODE_BASEBAND_TEST 

The test signal is routed to the input of the baseband filters instead of the RX mixer output signals.

IFX_TEST_SIGNAL_MODE_TOGGLE_TX_ENABLE 

TX1 is toggled on and off controlled by the generated test signal.

IFX_TEST_SIGNAL_MODE_TOGGLE_DAC_VALUE 

The power DAC value is toggled between minimum value and the value set trough ifx_avian_cw_set_tx_dac_value controlled by the generated test signal.

IFX_TEST_SIGNAL_MODE_TOGGLE_RX_SELF_TEST 

The test signal is routed directly into the RF mixer of RX antenna 1.
