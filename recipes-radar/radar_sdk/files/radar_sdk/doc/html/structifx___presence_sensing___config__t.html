Detailed Description

PresenceSensing configuration.

This structure contains all relevant parameters for creating an instance of the presence sensing module ifx_PresenceSensing_t, and some parameters which determine the operation of the presence sensing module.

You can use ifx_presence_sensing_get_config_defaults to get a default configuration that can serve as a starting point.
