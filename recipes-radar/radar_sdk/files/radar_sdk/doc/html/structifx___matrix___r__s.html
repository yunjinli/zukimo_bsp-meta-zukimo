Detailed Description

Defines the structure for a two-dimensional floating point data array.

Use type ifx_Matrix_R_t for this struct. Data length is fixed in this matrix i.e. matrix neither grows nor shrinks. The data is arranged sequentially in a row-major order (C order), i.e., all elements of a given row are placed in successive memory locations, as depicted in the illustrations.
