# NNA Firmware for DCT Detection Demo

The file mwcDemo.elf is the firmware for the DCT detection demo from the
Zuse KI mobil project. It may ONLY be used in the context of that project
because of license issues of the data used to train the contained CNN.


## Dependencies

* [Eigen](https://eigen.tuxfamily.org/) - [MPL 2.0](LICENSE-mpl-2.0.txt)
